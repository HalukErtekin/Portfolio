Bu siteyi sifirdan fork edip kendime uyarlarken amacim hem modern bir landing sayfasi, hem de uzun vadede blog ve paylasim alani olusturmakti.

## Ozellikler
- Next.js 13 App Router ve server component mimarisi
- Contentlayer ile MDX tabanli proje ve yazi icerigi
- Tailwind CSS ve lucide ikonlariyla hafif, koyu tema tasarimi
- Revalidate ile static/dinamik icerik dengesini koruyan mimari

## Komsal Servisler
- Vercel uzerinde CI/CD pipeline
- Upstash Redis yerine lokal counter (ileride analitik eklenecek)
- Cloudflare ile global CDN ve SSL

## Ogrendiklerim
- Tasarim dosyasi olmadan kendi UI kurma disiplini
- React Server Components ile performans optimizasyonlari
- Turkiye odakli bir portfolyonun hangi metriklere dikkat etmesi gerektigi

Siteyi gelistirmeye devam ediyorum; yeni projelerle beraber blog bolumunu da aktif hale getirecegim.
